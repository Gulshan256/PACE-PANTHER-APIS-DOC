
## Fetch Positions Daywise

| Request Type | APIs    | Endpoint                               | Description                     |
|-------------- | ------- | -------------------------------------- | --------------------------------- |
| `GET`        | Fetch Positions Daywise | https://pacetrader.pacefin.in/api/v1/positions?client_id=<ClientId>&type=live | Fetch Positions Daywise |


### Headers
| Header | Value |
|-------------- | ------- |
x-device-type | WEB
x-auth-token | (Your authorization token)
content-type | application/json


### Request Parameters

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| client_id | string | ClientId |
| type | string | live |



### Request Body
```json
{
    "client_id": "<ClientId>",
    "type": "live"
}
```
