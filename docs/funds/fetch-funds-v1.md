# Funds

## Fetch Funds V1

This API is used to fetch Funds. The query params are client id and type is all. In response the V1 fund's data like id, title, message etc. is provided. In case of error, It shows `Request Unauthorised`.



## Request
| Request Type | APIs    | Endpoint                               | Description                     |
|-------------- | ------- | -------------------------------------- | --------------------------------- |
| `GET`         | `Funds` | `https://pacetrader.pacefin.in/api/v1/funds/view?client_id=<ClientId>&type=all` | Fetches the user's funds. |


## Headers
| Header | Value |
|-------------- | ------- |
x-device-type | WEB
x-auth-token | (Your authorization token)
content-type | application/json



## Request Parameters
| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `client_id` | `string` | ClientId `<clintid>` |
| `type` | `string` | Type `<all>` |



## Request Body
```json
{
    "client_id":"<ClientId>",
    "type":"all"
}
```

## Response
```json
{
    "data": [
    {
    "id": "all",
    "title": "ALL"}]
    "message": "",
    "status": "success"
}
```

## Error Response
```json
{
    "status": "error",
    "message": "Request Unauthorised",
    "error_code": 40000,
    "data":{
    }
}
```
